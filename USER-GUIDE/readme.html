Kotakin
Version 1.2.5

First Things First

Kotakin was created for everyone that must constantly send files to customers or receive files from the same customers.

The employee can upload files from the administration system and share it with it’s customers or create a public link for anyone.

Every customer can have it’s own URL with a full screen background image and login system. In this area, the customer can upload files directly or download files provided by you.

Features :

  • Completely template-based for easy design modifications and object oriented template variables. Using Twig template engine.
  • Multilanguage.
  • Created with PHP ( Laravel 4 ) and the amazing Twitter Bootstrap version 3 – easy to modify and to understand.
  • Full Source Code.
  • SMTP / SENDMAIL / PHP MAIL Support.
  • Files renamed for security reasons. Original filename restored while dowloading.
  • Easy to use Step-By-Step installer.
  • Google Analytics Support.
  • Send Files directly by Email as Attachment.
  • Create Public Links ( with / without Password Protection, Download Limit, Date Limit, Message ).
  • Share Folders with registered Users.
  • Search for Folder and Files.
  • Beautiful Ajax Upload.
  • Superadmin / Admin / User roles.
  • Dateformat Settings.
  • Email Template Editor.
  • Preview Thumbnails.
  • GD and ImageMagick Support.
  • Direct Download of files.
  • Fullscreen Background can be set for each User separately.
  • Each User get’s it’s own URL like myappweb.com/username ( can be defined for each user ).
  • Each User has it’s own Login Area.
  • Users can upload Files directly to the Company.
  • Superadmin/Admin informed by Email of new Files Uploaded by a User.
  • User upload can be disabled.
  • Interface (HTML, CSS, and Javascript) Editor in browser.
  • Secure Public Link like myappweb.com/i/157b2102-ee92-4ce2-a715-9d6c1e07fc6d
  • Public Link Download Counter.
  • Auto disable Public Link if limits occur ( Download Counter, Date Limit ).
  • Console Commands that can be run as your server cron job to remove any expired and download limit reached links, with file (optional).
  • Disk usage capacity at admin dashboard.
  • Option to disable users upload in shared folder.
  • MP3 Preview (using http://kolber.github.io/audiojs/)
  • MP4 Video Preview (using http://www.videojs.com)

Installation:

  1. Extract the content of kotakin.zip file to your domain / subdomain directory, in example "/path/to/mysite.com".

  2. Hosting that enable you to change the web directory:

    Log in to the control panel and navigate to the Domain manager. Edit the domain where you wish to have Kotakin run from. Change the web directory from the default /path/to/mysite.com to /path/to/mysite.com/public. The reason we do this is that some of Laravel lives outside of the root web directory. Next we need to make sure PHP is set to 5.3.x or higher.

    Hosting with no access to change the web directory:

    Move all folders and files inside the "public/" folder to the root of your web directory.

    Now we need to edit our paths in index.php:

    require __DIR__.'/bootstrap/autoload.php';

    $app = require_once __DIR__.'/bootstrap/start.php';

    And then just set the public dir in out /bootstrap/paths.php file:

    'public' => __DIR__.'/..',

  3. Go to the domain / subdomain where the Kotakin installed via web browser.
  4. Follow and complete the installation steps.

Console Commands (cron jobs):

You can use following console commands manually or place them as your server cron jobs.

Go to the root of Kotakin application, where you can find the "artisan" file in there, then run following commands as you prefer.

  1. Remove any expired link, with the file (optional) 
    php artisan kotakin:removeExpiredLink
    php artisan kotakin:removeExpiredLink --with-file="true"
  2. Remove any download limit reached link, with the file (optional) 
    php artisan kotakin:removeDlLimitedLink
    php artisan kotakin:removeDlLimitedLink --with-file="true"

Localization (Language):

  1. Download the Poedit from http://www.poedit.net/.
  2. Edit file "workbench/sule/kotakin/src/lang/kotakin.pot" file using a text editor.
  3. Change the all the "msgstr" values with language translation, leave the "msgid" values.
  4. Save the file.
  5. Choose "New catalog from POT file" from Poedit and select kotakin.pot file.
  6. Fill required input on the Settings dialog that appear.
  7. When the save as dialog appear, fill the file name "kotakin.po".
  8. The Poedit will automatically create es_ES.mo file.
  9. Create a new folder "es_ES/LC_MESSAGES" inside "workbench/sule/kotakin/src/lang" folder.
  10. Save / Move the saved "kotakin.po" and "kotakin.mo" file to previously created folder.
  11. Go to to Admin preference and change the language to "es_ES"..
